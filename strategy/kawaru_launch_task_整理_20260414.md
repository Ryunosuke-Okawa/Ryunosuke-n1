## LINE動線構築：決めるべきこと

### Step 1: 現状把握（大川が確認）
- エヌイチ法人LINEの現在の友だち数・既存配信内容
- Lステップは導入済みか？既存のタグ・シナリオはあるか？
- 百奏音・大進がどこまで触れる状態か

### Step 2: 設計（大川が作成 → 大進に渡す）

**流入経路タグ設計**

| 流入元 | タグ名案 | 備考 |
|---|---|---|
| YouTube | `utm_youtube` | 概要欄リンク |
| Instagram | `utm_instagram` | プロフィール/ストーリーリンク |
| X | `utm_x` | プロフィール/固定ポスト |
| TikTok | `utm_tiktok` | ライブ経由 |
| 取りこぼしセミナー | `utm_seminar` | セミナー内告知 |
| LP直接 | `utm_lp` | LP内LINE登録ボタン |
| PRタイムズ | `utm_pr` | プレスリリース経由 |

**登録時アンケート項目**
- メールアドレス（必須）← Kawaru側との突合キー
- 電話番号（必須）
- 企業名（任意）← 法人識別用
- 役職（任意）← 決済者判別用

**自動配信シナリオ（Lステップ）**

```
登録直後 → 特典配布（リリース記念プレゼント）+ お礼メッセージ
1日後   → Kawaru LP案内 + 2週間無料トライアル訴求
3日後   → Kawaru活用事例紹介（個人ユースケース）
5日後   → セミナー案内（週1グルコン）
7日後   → ワークフロープレゼント紹介
14日後  → 未登録者にリマインド
```

**決済後の分岐**
- LINE未登録者 → サンクスページでLINE登録誘導
- LINE登録済み → Lステップタグで「トライアル中」に切替、CS系配信に移行

### Step 3: 特典の具体化
候補：
- 既存の「生成AI基礎研修ガイド」（一次特典としてあり）
- 業種別AIワークフローテンプレート集（Kawaruの価値を体感させる）
- 「AI導入チェックリスト」（法人向け訴求にもなる）

### Step 4: 大進への依頼書作成
上記をまとめて「LINE構築依頼書」として大進に渡す。
