# Kawaru 全SNS販売戦略 叩き台
**作成日**: 2026-04-04
**作成者**: 大川龍之介（PM）
**目的**: 2026年5月 Kawaru SaaS betaローンチに向けた、SNSを起点とした定常的B2Bリード獲得戦略の方針合意

---

## 0. エグゼクティブサマリー

Kawaru SaaSの販売においては、**SNSを起点に定常的な商談パイプラインを構築する**ことを目標とする。

現状は展示会・交流会・リファラルというスポット施策に依存しており、再現性のある定常リードがない。当社が保有するSNS運用ノウハウ（YouTube・X・Instagram）を活用し、**YouTubeを主軸とした法人向けリード獲得ファネルを設計・実行する**。

マーケティングのスコープは「商談創出まで」。その先のクロージングはセールスが担う。

---

## 1. 現状分析

### 1-1. 現在のリード獲得施策と課題

| 施策 | 現状 | 課題 |
|------|------|------|
| 交流会（高流界等） | リード獲得できている | スポット依存・マンパワー消費 |
| 展示会 | 直近でリード急増 | 担当者レベルが多く決裁者に届きにくい |
| リファラル | 最もCV率が高い | スケールしない |
| SEO | 月数十万のコスト | 白書DL 1〜2件/月・商談1〜2件/月・受注ゼロ |
| SNS（X→法人転用） | 試みたが中断 | 個人向け文脈が強く法人への転換率が低い |
| 法人YouTube | チャンネルあり数十再生 | ツール説明になっており全くCV出ず |

**本質的な課題**: 個人向けSNSインフラは強いが、**法人の意思決定者に定常的に届くチャンネルが存在しない**。

### 1-2. 当社の強み（活用できる資産）

- YouTube・X・Instagramの運用ノウハウと制作体制
- 演者として起用できる既存YouTuber（チャンネル転用想定）
- 法人向け実績（キリングループ・ミズカラ 月283h→85h削減等）
- Kawaruシリーズ4サービスのナレッジ・事例

## 2. 戦略コンセプト

### 「SNSで教育し、LINE経由で商談を作る」

```
認知・教育フェーズ          リード獲得フェーズ        商談化フェーズ
─────────────────────────────────────────────────────────────
YouTube Long動画            LINE登録                  架電
YouTube Shorts         →   アンケート回答        →   ウェビナー参加  →  商談
X（補完）                  電話番号取得              個別フォロー
```

- **YouTube**: 業種別・課題別のAI活用コンテンツで、課題を持つ担当者を教育しファン化する
- **LINE**: ファネルの核。リードを獲得し、電話番号取得まで設計する
- **ウェビナー**: 月1回の定期開催。参加者を商談へ転換する

## 3. チャンネル別戦略

### 3-1. 法人向けYouTube【主軸】

#### チャンネル設計方針

| 項目 | 内容 |
|------|------|
| チャンネル形態 | **演者型**（公式チャンネルではなく人にファンをつける） |
| 既存チャンネル | 既存YouTuberのチャンネルを法人向けに転用 |
| 演者 | 月曜MTGで決定（チャンネル方向性と合わせて選定） |
| ベンチマーク | 識学チャンネル（コンサル会社がYouTube思想発信→3,500社導入） |

**なぜ演者型か**: 公式（企業名）チャンネルはすでに認知がある企業でないと伸びない。「この人が面白い→このサービスを使いたい」という人→信頼→サービスの順番でリードを作る。

#### コンテンツ3本柱

**① 業種別・課題別AI活用シリーズ（主力）**

```
業種軸:
- 建設業×AI業務自動化
- 製造業×AI業務自動化
- バックオフィス×AI自動化（経理・人事・総務）
- 士業（税理士・社労士）×AI活用
- 小売・EC×AI活用

職種軸:
- 経理担当者のAI活用
- カスタマーサクセスチームのAI活用
- 営業事務×AI自動化
```

視聴者が「自分の業種・職種だ」と感じることで視聴継続率が上がり、ロングテール検索流入も生まれる。

**② AIニュースのビジネスインパクト解説（集客・認知）**

「新モデルが出た」ではなく「このニュースが企業経営にどう影響するか」の視点。

```
例:
「○○がAI活用で人員削減した背景を解説」
「生成AI活用率が15%止まりな日本企業に何が足りないか」
「最新のAI規制が中小企業に与える影響」
```

時事ネタは拡散・検索されやすく、新規視聴者獲得のフックになる。

**③ 課題解決デモシリーズ（CV直結）**

「この業務課題をKawaruで解決してみた」という実演形式。チャンネルが育ったタイミングでCTAと組み合わせてCVを促進する。

#### 動画フォーマット

| コンテンツ種別 | 推奨尺 | 投稿頻度（目安） |
|--------------|--------|----------------|
| 業種別AI活用 | 15〜20分 | 週1本 |
| AIニュース解説 | 10〜15分 | 週1本 |
| 課題解決デモ | 7〜12分 | 月2本 |

#### CTA設計

- 動画内CTA：「【無料】業種別AI活用チェックリストをLINEで受け取る」
- 動画説明欄：LINE登録リンク固定
- エンドカード：関連動画＋LINE登録ボタン

---

### 3-2. YouTube Shorts【補完・YouTube内誘導】

#### 役割

Long動画の切り抜きをShorts化し、**YouTube Shortsからロング動画へ誘導**する。
InstagramへのShorts展開はリソース観点から今フェーズでは行わない。

#### 運用方針

- Long動画1本につき1〜2本のShortsを切り出す
- Shortsは「一番刺さる1分」を抜粋（冒頭そのままではなく編集判断が必要）
- CTA：「詳しくはチャンネルのロング動画で」と誘導

---

### 3-3. X（旧Twitter）【補完・認知・YouTube誘導】

#### 役割

YouTubeへのトラフィック誘導と、経営者・AI推進担当へのブランド認知補完。
X単体でのリード獲得は狙わない。

#### 運用方針

- 奥山さん（CEO）の@okuyama_ai_アカウントを活用
- 個人向けAI発信の文脈を維持しながら、週1〜2本「法人向けAI活用テーマ」の投稿を混ぜる
- YouTube動画公開時に必ずXで告知・引用ポスト
- ウェビナー開催前1週間は集客告知を投稿

---

### 3-4. Instagram【今フェーズは動かさない】

個人向けB2Cコンテンツとして現状維持。
法人向けKawaru SaasのShortsをInstagramに流す工数と効果が見合わないと判断。将来的に見直し検討。

## 4. リード獲得ファネル設計

### 4-1. 全体フロー

```
【認知・集客】
YouTube Long動画 / YouTube Shorts / X告知
        ↓
【リード獲得】
CTA：「○○プレゼント（B2B向け特典）をLINEで受け取る」
        ↓
LINE登録 → 特典（プレゼント）配布
        ↓
【リード育成・情報取得】
「追加の特典（事例集・診断等）を受け取るにはアンケート回答を」
アンケート回答 → 企業名・役職・課題・電話番号 取得
        ↓
【架電】
架電リストに追加 → 架電でウェビナー誘導
        ↓
【商談化】
ウェビナー参加 → 個別相談・デモ → 商談
```

### 4-2. B2B向けプレゼント設計（重要）

現行のB2C向け特典をそのまま使うとリードの質が下がる。**特典のテーマをB2B文脈に変える**。

| 特典種別 | 内容 | 目的 |
|---------|------|------|
| 一次特典（LINE登録）| 業種別AI活用チェックリスト / AI導入失敗パターン解説PDF | B2B関心層を集める |
| 二次特典（アンケート回答）| Kawaru導入事例集（定量データあり）/ 無料AI業務診断 | 企業情報・電話番号取得 |

### 4-3. ウェビナー設計

| 項目 | 内容 |
|------|------|
| 開催頻度 | 月1回（定例固定）|
| 形式 | オンライン（Zoom） 45〜60分 |
| テーマ | 「AIで○○業務を自動化する実践セミナー」（テーマは月ごとに変える）|
| 登壇者 | 大川（現状最適）/ 岸田さん入社の場合は要検討 |
| CTA | ウェビナー後に「個別相談・デモのご案内」で商談へ誘導 |
| 集客経路 | LINE配信・X告知・YouTube動画内告知 |

## 5. KPI設計

### 追うべき指標（ビジネスインパクト順）

| フェーズ | KPI | 補足 |
|---------|-----|------|
| 商談 | 月間商談数（YouTube/SNS起点） | 最終アウトカム |
| ウェビナー | 月間参加者数 / 商談転換率 | 商談の母数 |
| リード | 月間LINE登録数 / アンケート回答率（電話番号取得率） | ウェビナーの母数 |
| 集客 | YouTube月間視聴数 / LINE登録転換率 | リードの母数 |

**チャンネル登録者数・再生数は直接KPIにしない**（B2Bリード獲得との相関が低いため）。

## 付録：なぜYouTubeか（議論用メモ）

### X・Instagram・YouTubeの役割比較

| | X | Instagram | YouTube |
|--|---|-----------|---------|
| 使われ方 | 受動的に情報収集 | 受動的に発見 | **能動的に課題の解決策を探す** |
| コンテンツ寿命 | 数時間 | 数日 | **数年単位で機能し続ける** |
| デモ・説明の深さ | 浅い | 浅い | **深い（15〜20分で教育できる）** |
| 資産蓄積 | ほぼなし | 弱い | **動画が増えるほどリードが増える** |
| B2B検索流入 | なし | なし | **あり（世界2位の検索エンジン）** |

### ベンチマーク：識学チャンネル

コンサルティング会社がYouTubeで「マネジメント思想」を発信し続けた結果、導入企業3,500社超・書籍累計184万部に成長。「ツール説明」ではなく「課題解決の思想」を発信するスタイルは、エヌイチがKawaruで目指す方向と構造が同じ。
