# 法人トスアップ用 事前ヒアリングテンプレ v1

> [!info] 目的
> スクール事業部側の営業マン（前田さん 等）が、商談中に法人案件の芽を検知した時、
> 大川（法人担当）に引き継ぐ前に**先方から最低限・簡易に聞いておいてほしい項目**。
>
> - 先方の負担を最小化（書面・3分以内で答えられる粒度）
> - 大川側がMTG価値を一次判定できる情報量を確保
> - 業種・従業員規模など**会社名から調べられる情報は除外**

---

## 質問リスト（先方に送付／口頭で聞き取る）

```
1. お名前
2. 会社名
3. ご所属部署・お役職
4. メールアドレス
5. 電話番号
6. ご自身のミッション（社内で担われている役割）
7. AI活用で解決したい、具体的な業務上の困りごと
8. 今回AI活用に関心を持たれたきっかけ
   （ご自身の関心 ／ 会社・上長からの依頼 ／ 両方）
```

---

## 各項目の意図（営業マン共有用メモ）

| 項目 | 意図 |
|---|---|
| 1 お名前 | フォローアクションの起点 |
| 2 会社名 | これさえあれば**業種・規模・事業内容はこちらで調べられる** |
| 3 部署・役職 | championか決裁手前かの一次判定 |
| 4 メアド | 顔つなぎMTGの調整動線 |
| 5 電話番号 | メール往復が遅い時の電話一発手段 |
| 6 ミッション | AI活用がこの人の役割に含まれるか／個人発信かの判定材料 |
| 7 業務上の困りごと | **Painの具体性**で本気度がわかる（薄い＝個人関心／具体的＝champion） |
| 8 きっかけ | **個人関心／会社指示／両方**＝商談温度感と進行速度が決まる |

---

## 設計上の意思決定ログ

- **業種・事業内容／従業員規模**は会社名から調べられるので外した（先方の手間優先）
- **「困りごと」と「実現したいこと」を1問にまとめる案は不採用**。Painだけに絞った方が答えが具体的になる。「理想・未来像」は法人MTGで本人から直接聞く方が深い
- **観察事項（営業マン側のメモ）は不要**と判断。質問リストのみで運用
- **会社のミッション**は外した（HPで見れば分かる／先方が即答に困る）。代わりに**本人個人のミッション**を聞くことで「個人発信 vs 会社発信」の判定軸になる
