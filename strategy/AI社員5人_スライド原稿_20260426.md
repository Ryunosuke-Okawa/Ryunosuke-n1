## スライド1：Kawaru事業部は「コストダウン」の話ができない（10秒）

### キーメッセージ
固定業務がないから、削減対象がない

### 表示内容

- **大見出し**：私たちは "コストダウン" の話ができません
- **サブ見出し**：Kawaru事業部の前提
- **対比ビジュアル**（白×紺）：

```
[既存事業]              [Kawaru事業部]
固定業務がある    ←→   業務が毎月変わる
削減対象が見える       削減対象がそもそもない
```

### 話す原稿（10秒・約45文字）

> 私たちは、コストダウンの話ができません。  
> 削減する元の固定業務が、毎月変わるからです。

### 補足ノート
- 否定から入って聞き手の期待値を整える
- 「だから何をするのか」の伏線
- 髙橋COO含め全員が共有している前提を確認する役割

### キーメッセージ
プロダクト作りをフォーカス、3つのデリバリーサービスは同時進行

### 表示内容（4サービスカラー使用）

- **メインビジュアル**：

```
       ┌──────────────────────┐
       │  Kawaru プロダクト作り  │  ← 青（フォーカス強調）
       │      ★ フォーカス ★      │
       └──────────────────────┘
                  ↓
            AI社員を動かす

         ─ 同時進行 ─

   [Kawaru Team]  [Kawaru Coach]  [Kawaru BPO]
    オレンジ        紫              緑
       3つのデリバリーサービスも動かす
```

- **小さく注記**：「プロダクト作りとデリバリーは同時進行、フォーカスはプロダクト作り」

### 話す原稿（15秒・約70文字）

> 代わりに私たちは、AI社員を動かして、まず **Kawaruプロダクトの作り込み** を前に進めます。同時に **Kawaru Team・Coach・BPOの3つのデリバリーサービス** も動かします。

- プロダクト作りを独立した中央フォーカスとして配置
- デリバリーは下に3サービス並列で配置
- カラーで4サービスの差を表現（このスライドは4サービス一覧扱い）
- 「動かす」の能動性を強調

## スライド3：AI社員5人の構成（25秒）

### キーメッセージ
4人は各領域の戦略+実行、戦略AIは全体のリサーチ・分析担当

### 表示内容（白×紺×アクセント青）

- **タイトル**：AI社員5人の構成
- **5人を横並びで配置**：

| AI社員 | 担当領域 |
|-------|---------|
| **マーケAI** | LP／SNS／展示会／LINE配信／事例記事 |
| **営業AI** | 商談前準備／要件ヒアリング／提案書／フォロー |
| **デリバリーAI** | 研修運営／Dify・N8N・GAS構築／CSコンサル／PM |
| **事務・法務AI** | 契約／請求／PL／利用規約 |
| **戦略・プロダクトAI** | 競合・市場・料金・データ分析（**リサーチ・分析担当**） |

- **下部キャプション**：「マーケから事務まで4人 = 各領域の戦略 + 実行／戦略・プロダクトAI = 全体のリサーチ・分析担当」

### 話す原稿（25秒・約115文字）

> AI社員5人の構成です。**マーケ・営業・デリバリー・事務法務・戦略プロダクト**。  
> マーケから事務までの4人は、**それぞれ自分の領域の戦略と実行**を持ちます。  
> 戦略プロダクトAIは、**全体のリサーチと分析**を担当し、4人を支えます。

### 補足ノート
- 戦略AIを「リサーチ・分析担当」と明示（「横断支援」は使わない）
- 各AIが自領域の戦略を持つことを強調（指示待ちではない）
- 5人の役割が一目で分かる横並び構造

## スライド6：半期¥6,710,000、4案件すべてにAI社員が動いている（25秒）

### キーメッセージ
売上関与＝もう動いている。月¥464,000＝AI社員1.3人分稼働中

### 表示内容（白×紺×アクセント青）

- **タイトル**：半期¥6,710,000、4案件すべてにAI社員が動いている
- **メイン表**：

| 案件 | 売上（税抜） |
|-----|-----------|
| NTTネクシア | ¥5,000,000 |
| 中西製作所 | ¥1,000,000 |
| 中部キリンビバレッジ | ¥450,000 |
| ブロードリンク | ¥260,000 |
| **半期合計** | **¥6,710,000** |

- **右下強調枠**（青）：

```
月換算 ¥464,000
= AI社員 1.3人分が稼働中
```

### 話す原稿（25秒・約115文字）

> 10月から3月の半期で売上 **¥6,710,000**、4案件すべてにAI社員が動いています。  
> NTTネクシア、中西製作所、キリン、ブロードリンク。  
> 月換算で **¥464,000**、**AI社員1.3人分がすでに稼働しています**。

### 補足ノート
- 数字で説得力
- 「すべての案件に関与」 = 網羅性
- 「机上ではない」現実性の証明

## スライド7：上流業務でAI社員を使い倒している（25秒）

### キーメッセージ
プロダクト作りもデリバリーも、上流業務でAI社員が動いている

### 表示内容（4サービスカラー使用）

- **タイトル**：上流業務でAI社員を使い倒す
- **2列構成（プロダクト作りを先に）**：

**【Kawaru プロダクト作り】** ← 青で強調

| 業務 | 担当AI社員 |
|-----|----------|
| プロダクト改善の提案たたき | 戦略AI（リサーチ・分析担当） |
| ローンチPM・LP・LINE配信 | マーケAI＋戦略AI |
| 競合・料金・データ分析 | 戦略AI |

**【Kawaru Team / Coach / BPO のデリバリー】** ← 各サービスカラー

| 業務 | 担当AI社員 |
|-----|----------|
| 展示会設計・クリエイティブ | マーケAI |
| 提案書骨子作成 | 営業AI |
| Dify／N8N／GAS構築 | デリバリーAI |

- **下部強調**：「時給3,000〜5,000円帯の上流領域でAIを使い倒している」

### 話す原稿（25秒・約115文字）

> AI社員は売上に関わるだけではありません。  
> **Kawaruプロダクトでは**、改善提案・ローンチPM・データ分析。  
> **Kawaru Team・Coach・BPOでは**、展示会設計・提案書・Dify構築。  
> **時給単価が高い上流領域でAIを使い倒しています**。

### 補足ノート
- プロダクト作りを左（先）に置く
- デリバリーを右に
- 時給3,000〜5,000円帯を強調（業務効率化の話ではないことを示す）
- このスライドも4サービスカラーOK

## スライド8：6月末2人分→9月末5人分、Agent Teamsも並走（25秒）

### キーメッセージ
5人を網羅的に同時進行で育成、Agent Teamsも並走で構築

### 表示内容（白×紺×アクセント青）

- **タイトル**：6月末2人分→9月末5人分、Agent Teamsも並走で構築
- **ロードマップ表**：

| 月 | 達成水準 | 動き |
|---|---------|-----|
| 4月（現在） | 1.3人分 | デリバリーAI＋戦略AI が稼働中 |
| **6月末（中間発表）** | **2人分** | 5人全員が同時進行で立ち上がり／**Agent Teams 初期構築** |
| 7-8月 | 3〜4人分 | 各領域で深化／Agent Teams 連携テスト |
| **9月末（最終）** | **5人分** | **5人フル稼働 + Agent Teamsで協働** |

- **下部メッセージ**：「特定の1人を完成させるのではなく、5人を網羅的に育てる」

### 話す原稿（25秒・約115文字）

> 6月末までに **2人分**、9月末までに **5人分** まで育てます。  
> 1人ずつ完成させるのではなく、**5人を網羅的に同時進行で育てる** やり方です。  
> 同時に **Agent Teamsの仕組みも並走で構築** し、9月末には5人がチームとして協働する状態を作ります。

### 補足ノート
- 「同時進行・網羅的」を強調（現実的な育成方針）
- Agent Teams並走を明示（5人完成後ではない）
