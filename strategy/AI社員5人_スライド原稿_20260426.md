## スライド1：Kawaru事業部は「コストダウン」の話ができない（10秒）

### キーメッセージ
固定業務がないから、削減対象がない

### 表示内容

- **大見出し**：私たちは "コストダウン" の話ができません

```
[既存事業]              [Kawaru事業部]
固定業務がある    ←→   業務が毎月変わる
削減対象が見える       削減対象がそもそもない
```

### 話す原稿（10秒・約45文字）

> 私たちは、コストダウンの話ができません。  
> 削減する元の固定業務が、毎月変わるからです。

### 補足ノート
- 否定から入って聞き手の期待値を整える
- 「だから何をするのか」という伏線
- 髙橋COO含め全員が共有している前提を確認する役割

## スライド3：AI社員5人の構成（25秒）

### キーメッセージ
機能別に5人、それぞれが自領域の戦略と実行を担う

### 表示内容

- **タイトル**：AI社員5人の構成
- **5人を横並びで配置**：

| AI社員 | 担当領域 |
|-------|---------|
| **マーケAI** | LP／SNS／展示会／LINE配信／事例記事 |
| **営業AI** | 商談前準備／要件ヒアリング／提案書／フォロー |
| **デリバリーAI** | 研修運営／Dify・N8N・GAS構築／CSコンサル／PM |
| **事務・法務AI** | 契約／請求／PL／利用規約 |
| **戦略・プロダクトAI** | 競合・市場・料金・データ分析（**横断支援**） |

- **下部キャプション**：「マーケから事務まで4人 = 各領域の戦略 + 実行／戦略AI = 横断リサーチ・分析」

### 話す原稿（25秒・約115文字）

> AI社員5人の構成です。**マーケ・営業・デリバリー・事務法務・戦略プロダクト**。  
> マーケから事務までの4人は、**それぞれ自分の領域の戦略と実行**を持ちます。  
> 戦略プロダクトAIは、**横断的にリサーチと分析**を担当し、4人を支えます。

### 補足ノート
- 戦略AIを「上から降ろす」ではなく「横断支援」として明示
- 各AIが自領域の戦略を持つことを強調（指示待ちではない）
- 5人の役割が一目で分かる横並び構造

## 想定Q&A（発表後の質疑応答に備えて）

### Q1：AI社員1人=月35万円分の根拠は？
時給1,500円（事務・作業系）／3,000円（提案・調査）／5,000円（戦略）のミックスで月140時間程度。会社側の時給設定が確定したら更新。

### Q2：6月末で2人分は本当にいけるのか？
4月時点で1.3人分が稼働中。5月にKawaruローンチ準備で5人全員が同時進行で立ち上がる。各AI社員の稼働深化で2人分は射程内。

### Q3：他のメンバーが同じことをやるには？
Vault（Obsidian）と Claude Code Skill群が共有資産。スキル＋Vaultがあれば他メンバーも自分専用AI社員を立ち上げられる。

### Q4：CSやデータ分析は？
CS は ③デリバリーAIに内包（Kawaru組織プランのAIコンサルタント＝デリバリー扱い、パーソナルプランはCSなし）。データ分析は①②⑤に分散内包。9月末以降に必要なら独立可能。

### Q5：Kawaruプロダクトへの直接貢献は？
戦略AIが競合・料金・ログ分析でプロダクト改善提案を支援。マーケAIがLP/SNS/LINEで集客に直結。営業AIが商談・成約に直結。

### Q6：Agent Teamsとは？
複数のClaude Codeエージェントに役割を与え、並列で議論・実行させる仕組み。5人のAI社員がチームとして横断連携する状態を9月末に作る。
