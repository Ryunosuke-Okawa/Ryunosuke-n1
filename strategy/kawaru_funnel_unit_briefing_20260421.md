# Kawaru β版ローンチ LINE動線 — ファネルユニット向けブリーフィング

> [!info] 目的
> 4/14タスク整理で4/20期日としていた「ファネルチームブリーフィング」を実施。百奏音さんに構成・全体感のたたきを依頼、その後大進さんに構築を引き渡す。今週中（〜4/24）にフローチャート・全体感のたたきを受領することを目標とする。

---

## 0. 役割分担（前提）

| 担当 | 役割 | 期待アウトプット |
|------|------|----------------|
| **百奏音** | 構成・全体感 | LINE登録〜CTA〜継続配信のフローチャート／媒体別動線／訴求文言骨子 |
| **大進** | 実際の構築 | Lステップ設定／流入経路タグ／自動配信シナリオ／アンケート |
| **大川** | PM・素材提供・管理 | 特典制作・台本・LP・計測仕組み検証 |

流れ：**百奏音が構成を切る → 大川が確認 → 大進が構築**

---

## 1. 絶対条件（4/20MTG決定）

| 項目 | 内容 |
|------|------|
| ローンチ日 | **2026-05-26（火）** |
| KGI | **無料登録500名**（期間：5/26〜6月末） |
| 大枠配分 | SNS 400名（YouTube 286 / Instagram 1,714 / X 857） + セミナー100名 |
| 通算転換率 | LINE登録 → LP閲覧 **70%** → Kawaru登録 **20%** ＝合計 **14%** |
| Kawaru専用LINE | Bちゃん既存HPに集約（松浦指揮／宇部構築／4/25までに完成） |
| 決済 | VISA/Master両方通過見込み（Amazon Pay不可） |
| 価格訴求 | 2週間無料トライアル推し／月額9,000円は動画内NG |

## 2. 動線の全体像（媒体別マトリクス）

### 導線A：SNSローンチ（外向き）

| 媒体 | 担当 | ローンチ期間（5/26-5/31） | 通常運用 |
|------|------|---------------------|---------|
| **X** | 松浦 | Kawaru専用LINE直接 | **常時Kawaru専用LINE直接** |
| **YouTube** | 河村 | Kawaru専用LINE直接 | **基本Kawaru専用LINE直接**（プレゼント設計次第で二段階も可） |
| **Instagram** | 今くん | Kawaru専用LINE直接 | **ハブLINE→サンクスページ→Kawaru LINE**（ハブ登録率維持） |
| **TikTok** | 近 | ライブ集客 | — |

### 導線B：ハウスリスト（1万件・主力）

- 5/26〜5/31にLINE一斉配信＋LP＋活用映像
- 訴求：「Claude Codeに疲れたあなたに」系（建材ユーザー等の機能的ユーザー優先）

### 導線C：セミナー（内向き・取りこぼし回収）

- 5/28夜〜5/31（朝昼晩×4日＝**全12回**）
- AIエージェント道場と併催／登録100名目標
- セミナー内でLP訴求 → 画面共有で一緒に登録
- **未購入者 → Lステップ継続配信でフォロー**
- **失注者 → 資料請求ページ＋LINEフォローシナリオ**

### 共通ファネル（登録後）

```
Kawaru専用LINE登録
   ↓
LINE内でLP送付（紹介動画＝奥山・河村出演・10〜15分埋め込み）
   ↓
LP閲覧（70%）
   ↓
無料トライアル登録（カード登録・20%転換）
   ↓
サンクスページ（LINE未登録者→再誘導）
   ↓
Lステップ継続配信（週1セミナー案内／WF配布／活用事例）
   ↓
チームプラン転換（有料化）
```

## 3. 大川さんの認識ズレ（自己チェック済み・百奏音に正しい前提として共有）

| 大川さんが言いそうな前提 | 正しい理解 |
|---|---|
| ローンチ期間だけBLINEに流す | **Xは常時・YouTubeも基本常時・Instagramだけ期間切替** |
| セミナーやってない期間はハブLINEに落とす | **Instagramのみの運用**。X/YouTubeは常時Kawaru専用LINE直接 |
| セミナー参加者の流入元が不明 | **各SNSごとにLINE流入経路タグを付与**（4/13決定）→計測可能 |

## 4. 漏れ要素（百奏音のチャートに必ず入れる）

3. **LINE内教育動画**（河村撮影10-15分、台本は大川作成・期日4/25）
4. **プレゼント機能Bパターン**（ファイルインポート形式）＝LINE登録特典・セミナー特典
5. **Lステップ継続配信シナリオ**（週1セミナー案内／WFプレゼント／活用事例／チームプラン案内）
6. **UTMタグ＋LINE流入タグ設計**（計測・改善サイクル）
7. **Claude CodeチャンネルLINE**の扱い（当面Kawaru専用LINEと同一枠運用）

## 7. 既にあるたたき（百奏音さんに渡す素材）

4/14タスク整理Step2より、大川側で既に骨子を作成済み。百奏音さんはこれをベースに構成を切り直す想定。

### 流入経路タグ設計案

| 流入元 | タグ名案 |
|---|---|
| YouTube | `utm_youtube` |
| Instagram | `utm_instagram` |
| X | `utm_x` |
| TikTok | `utm_tiktok` |
| 取りこぼしセミナー | `utm_seminar` |
| LP直接 | `utm_lp` |
| PRタイムズ | `utm_pr` |
| ハウスリスト | `utm_houselist`（追加） |

### 登録時アンケート項目

- メールアドレス（必須）← Kawaru側との突合キー
- 電話番号（必須）
- 企業名（任意）
- 役職（任意）

### 自動配信シナリオ骨子

| タイミング | 配信内容 |
|---|---|
| 登録直後 | 特典配布（リリース記念プレゼント）＋ お礼メッセージ |
| 1日後 | Kawaru LP案内＋2週間無料トライアル訴求 |
| 3日後 | Kawaru活用事例紹介（個人ユースケース） |
| 5日後 | セミナー案内（週1グルコン／5/28-5/31の12回告知） |
| 7日後 | ワークフロープレゼント紹介（Bパターン） |
| 14日後 | 未登録者にリマインド |

### 決済後分岐

- LINE未登録者 → サンクスページでLINE登録誘導
- LINE登録済み → Lステップタグで「トライアル中」に切替、CS系配信に移行
