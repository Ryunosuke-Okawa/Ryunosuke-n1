## 不正と判断した根拠

1. 弊社が稼働していないベルギー発のAPIコール
2. 弊社アプリで使用していないモデル（`gemini-3.1-pro`・`gemini-2.5-pro` 等）で6,000万トークン以上を消費
   - `gemini-3.1-pro`：約4,000万トークン
   - `gemini-2.5-pro`：約1,900万トークン
   - `gemini-2.0-flash`：約140万トークン（4/6に異常スパイク）
   - `gemini-3.1-flash-lite` / `gemini-2.5-flash-lite` 等
3. Cloud Monitoringで `apikey:UNKNOWN` からの呼び出し319件、403エラー7,563件
4. 3月の通常利用と比較して404%の異常増加
5. 弊社稼働時間外の大量アクセス

## 実施済み対応（技術面）

- [x] 該当APIキーの削除（2026-04-21）
- [x] 新規キー発行（IP制限・APIターゲット制限付き）
- [x] `gemini-3-pro` に 10,000 tokens/min のQuota制限
- [x] Data Access監査ログの有効化
