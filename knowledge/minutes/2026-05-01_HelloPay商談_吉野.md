## 1. HelloPay サービス概要（髙橋氏より）

### 1-1. 立ち上げ経緯
- HelloPay運営グループは元々マッチングアプリ・予約・ヘルスケアサブスク等を運営し、**GMOペイメント／ソニーペイメント等の大手PSPを多用**。
- ただし「手数料が高い」課題があり、**原価で仕入れて自社で抜き幅を抑える＝既存決済からの切替インセンティブを作る**思想で内製化。
- すでにグループ会社で実装・稼働済みのため**1,000億規模を支える基盤を持つ**と訴求。

### 1-2. 価格訴求
- **業界最安水準の手数料**を提示可能（業種・リスク区分による）。
  - **ハイリスク商材**（情報商材・スクール・コーチング等の一部）: **4.5%程度**
  - 一般商材は減価ベースで個別見積。返金率や他社決済コストをヒアリングして算出。
- 初期費用・月額費用は基本ゼロ、**価格は「後出しじゃんけん」**（他社条件に合わせる）。
- 「手数料率が下がれば、売上はそのままでも利益が増える」が訴求の核。

### 1-3. プラン構成
- 一般業種: 月商で区切り（400万 / 2,000万以下）
- 他業種（マッチング／オンライン講座／工学／コーチング）は**事業ごとに個別決済プラン**
  - → エヌイチの場合、個人向け/法人向けでサービスが複数あるため、**サービス単位での見積分割**になる前提

### 1-4. 対応決済手段
| 区分 | 対応 |
|------|------|
| クレジットカード | VISA / Mastercard / JCB / AMEX |
| モバイル決済 | Apple Pay / Google Pay |
| その他 | コード不要の決済リンク、リンク決済、トークン決済 |
| 機能 | 売上管理ダッシュボード、Webhook通知、入金管理、サブスク設定、不正検知、チャージ枠管理 |

### 1-5. 入金サイクル
- 一般商材で**月2回**（他PSPと同水準）
- 商材ごとに個別設定

### 1-6. セキュリティ・コンプライアンス
- **クレカ協会の最新セキュリティ基準に準拠**
- **フレーム型SDK**による実装。グループ会社サービスで実装・運用実績あり。

### 1-7. 実績・既存導入
- グループ会社事例:
  - **ゴージャス**（マッチングアプリ）
  - **オートリザーブ**（予約管理SaaS、お任せ系）
  - **ヘルスケア系サブスク**（恵比寿のカロ社、**年間流通5億円規模**）
- 「いずれストライプ等の大型企業からの切替も狙っていく」と発言、ただし今は紹介ベース。

## 2. エヌイチ側 現状ヒアリング

### 2-1. 事業構成
- 2軸構成
  - **個人向け事業**: 5〜6サービス。動画 + チャットサポート + Zoom質問対応。
    - 例: 「ChatGPT道場」など複数
    - 期間別パターン: 1ヶ月完結 / 半年使用後サブスク化 / 1年プラン など
  - **法人向け事業**: 4プラン
    - 最廉価: SaaSツール（カードサブスク。月額固定）
    - 中位: 講師派遣、グループセッション、定期研修
    - 上位: 経営コンサル領域まで踏み込み
    - 一括（数十名規模、20名一括購入など）または月額サブスクで提供
- 個人向けはサブスク中心、法人向けは一括銀振が多い。最廉価法人プランのみカードサブスク。

### 2-2. 現在の決済構成・流通額

| 決済代行 | 月間流通（直近4月） | 比率 |
|----------|---|---|
| **UNIVAPAY**（メイン） | 約 **2,700万円** | **約82%** |
| **MOSH（モッシュ）** | 約 **50万円** | — |
| **CBS**（※文字起こしでは「CBS」と発話。詳細未確認） | — | **約10%** |
| その他（少額決済） | — | 数% |

- **UNIVAPAY条件**: 手数料率**3.6%**、トランザクション費なし、月額なし、ローリングリザーブなし。**手数料は顧客側に転嫁**。
- **カード比率は流通全体の約85%**（口座振替なし、銀振は法人一括のみ）

### 2-3. 直面している3つの課題

#### ① UNIVAPAYで新サービスのアカウントが取れない
- 既存サービスは継続利用可だが、**新規サービスを立ち上げるたびにUNIVAPAY上でアカウント発行ができない状況**。
- → 新規事業立ち上げ時の決済代行確保が常時ボトルネックになる。

#### ② 過去2〜3社で審査落ち（事業者リスト登録）
- 経緯:
  1. **2025年7月**: ユーザーからクレーム発生。返金は完了済だったが、**弁護士経由で再調査要請**。
  2. 営業担当による **契約書の読み合わせが未実施** だった点、**広告の打ち出しが強めだった**点が問題視される。
  3. **フィンコード（および他PSP数社）で保留／審査落ち**。
  4. **2025年12月**: 最終的に **事業者リスト** に登録され、以降の新規申込が通りにくくなった。
- → スクール系・情報商材系の業界全体での審査厳格化トレンドとも重なり、エヌイチ単体での突破が困難になっている。

#### ③ 口座振替を新たな決済手段として持ちたい
- サブスクプランで「カードを持たない／使えない」顧客の取りこぼし対応。
- 既存にはなく、HelloPayでの対応可否を確認したい。

## 3. HelloPay対応可否のすり合わせ

| 検討項目 | HelloPay回答 |
|----------|-------------|
| カード決済（VISA/Master/JCB/AMEX） | ✅ 即対応可 |
| サブスク決済 | ✅ 対応可 |
| **口座振替** | ⚠️ **現状は未対応**。希望が出てきたため、社内で導入可否・予定を確認 |
| 事業者リスト登録後の通過可能性 | アクワイアラ側でエヌイチのサービスが登録されているか／審査落ち理由を髙橋氏が照会。**「基本プランは通したい」スタンス** |
| 手数料水準 | UNIVAPAYの3.6%より**安くできる可能性が高い**。ただしハイリスク区分なら4.5%が下限。詳細はサービス別に見積 |

### サポート体制（吉野氏質問への回答）
- **他社と同等のサポート水準**を維持（安くなる代わりにサービス品質が落ちることはない）。
- 操作トラブル等はメール／電話／LINEで**即時対応**。
- HelloPay代表がエンジニア出身のため、**技術問い合わせも即レス可能**。これは他社優位になる可能性ありと自負。

### サインアップ・初期費用
- 初期費用・月額費用は基本ゼロ。
- ローリングリザーブ（売上の一部を一定期間預かる慣行）も**現状なし**と確認。

## 5. 大川追記：所感・戦略視点

### 5-1. 吉野さん単独参加の課題
- 商談中、**サービス内容や流通数値を即答できない場面が多発**。次回までに以下を整理して再臨むのが望ましい:
  - サービス別の月次流通額・サブスク/一括比率
  - 過去の審査落ちの正式経緯資料（弁護士指摘内容、広告表現の論点整理、改善対応の有無）
  - 口座振替を必要とする対象プラン（どのプランで、想定何件か、月いくらか）
  - 「CBS」と発話されている10%枠の決済会社の正体確認

### 5-2. HelloPay側の温度感
- 「**基本プランは通したい**」「**後出しじゃんけんで条件を合わせる**」と前向き発言が多く、**条件交渉余地は大きい**。
- ただし「スクール系の業界審査が厳しい」という業界トレンド自体は明言されており、**エヌイチ側の広告表現／契約フローの是正は前提条件**になる可能性が高い。
- 紹介経由かつ「みさん発のXで案件が増えている」状況なので、HelloPay側にも「通す＝実績を作る」インセンティブがある。

### 5-3. ディール最大の論点
- **口座振替の対応可否**。これが入れば「カード代替手段の確保」というエヌイチの戦略目的に直接刺さる。
- 一方、カード決済の置き換えだけだと、UNIVAPAYの**手数料3.6%／顧客転嫁／月額ゼロ**は条件として既に十分良好。
  - → **HelloPayが3.6%を下回り、かつ事業者リストを突破できる**ことが切替条件になる。
  - → ハイリスク区分（4.5%）扱いになると逆に不利。**ここの線引きを早期に確認したい**。

### 5-4. Kawaru事業部視点でのキープ価値（大川目線）
- Kawaru Team / Coach / BPOで将来、**法人サブスク決済**を組む際の候補としても押さえておく価値あり。
- Kawaruは法人向けかつ研修・コンサル文脈なので「ハイリスク商材」扱いを受けにくい可能性があり、HelloPayの一般商材レートが効きやすい領域。
- いずれにせよ **「奥山CEO発の紹介ディール」** なので、ライン管理は奥山にも共有しておくのが筋。
