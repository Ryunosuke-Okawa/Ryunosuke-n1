# PayPalサブスク決済設定ナレッジ

## 対象サービス

**Kawaru（カワル）** — エヌイチが提供するAIを活用した経営支援サービス。法人・個人向けのコンサルティング/サポートサービスとしてサブスクリプション課金を設定。

---

## 料金設計

### Kawaruパーソナルプラン

| 項目 | 設定値 |
|------|--------|
| プラン名 | Kawaruパーソナルプラン |
| 製品名 | Kawaru |
| 製品タイプ | サービス |
| 料金タイプ | 固定料金設定 |
| 通貨 | JPY |
| お試し期間 | 2週間（14日間）/ 無料（¥0） |
| 月額料金 | **¥9,900（税込）** |
| 請求サイクル | 毎月 |
| 請求期間 | 無制限 |
| 未払い自動請求 | オン |
| 税金計算 | なし（税込価格で設定） |

> **税金の扱い**: PayPalは税抜・税込の自動計算を行わない。当初は税抜¥9,000の想定だったが、消費税10%込みの**¥9,900**で設定する方針に決定。顧客には「税込¥9,900/月」として案内する。

## PayPalでのサブスク決済設定手順

### 前提条件

- PayPal**ビジネスアカウント**が必要（個人アカウントでは不可）

### 方法の選択肢

| 方法 | 難易度 | 向いてる用途 |
|------|--------|------------|
| **PayPalボタン（ノーコード）** | 簡単 | LPに貼るだけ — 今回採用 |
| Subscriptions API | 開発必要 | カスタマイズしたい場合 |

### 設定手順（PayPalボタン方式）

#### Step 1: PayPal管理画面にアクセス

1. PayPalビジネスアカウントにログイン
2. 左メニューの「**購読プラン**」をクリック
   - 「購読プラン」= サブスクプランの設定・作成
   - 「サブスクリプション」= 購読している顧客の一覧（別物）

#### Step 2: 製品を作成

| 項目 | 入力内容 | 必須/任意 |
|------|--------|----------|
| 製品名 | Kawaru | 必須 |
| 商品の説明 | AIを活用した経営支援サービス | 任意 |
| 製品タイプ | **サービス** | 入れておくと◎ |
| 業種カテゴリ | Software / SaaS | 任意 |
| 製品ページURL | LPのURL（後で変更可） | 任意 |
| 画像URL | Kawaruのロゴ画像URL | 任意 |

#### Step 3: プラン設定

| 項目 | 設定値 |
|------|--------|
| プラン名 | Kawaruパーソナルプラン |
| プランの説明 | 任意（管理用メモ程度でOK） |
| 通貨 | JPY |
| 1回限りの設定料 | なし |
| お試し期間 | オン |
| お試し期間の料金 | ¥0 |
| お試し期間のサイクル | 毎2週（1サイクル = 14日間） |
| 購読期間の料金 | ¥9,900 |
| 購読期間のサイクル | 毎1ヶ月 |
| 購読期間 | 無制限の請求サイクル |
| 税金計算ツール | 税金は計算しません |
| 未払い自動請求 | オン |

#### Step 4: プランの有効化

- 確認画面で内容をチェック → 「**プランをオンにする**」を押す
- 「下書きを保存」だとまだ使えない状態になるので、必ず「オンにする」を選択

#### Step 5: 埋め込みコードを取得

- プラン作成完了後、「**コードのコピー**」ボタンでHTMLコードを取得
- このコードをエンジニアにそのまま渡せばOK

## LP連携の方法

### 埋め込みコード（生成されるもの）

```html
<!-- PayPalが生成するサブスク決済ボタン -->
<div id="paypal-button-container-PLAN_ID"></div>
<script src="https://www.paypalobjects.com/web/res/..."></script>
<script>
  paypal.Buttons({
    createSubscription: function(data, actions) {
      return actions.subscription.create({'plan_id': 'P-XXXXXXXXXX'});
    },
    onApprove: function(data, actions) {
      alert('登録完了！ID: ' + data.subscriptionID);
    }
  }).render('#paypal-button-container-PLAN_ID');
</script>
```

### エンジニアへの引き渡し

- PayPal管理画面で「コードのコピー」で取得したHTMLコードをそのまま渡す
- LPの任意の場所に貼り付けるだけで動作する
- `plan_id`（`P-XXXXXXXXXX`）は自動的にコードに含まれる

### 対応LP環境

HTMLを貼り付けられる環境であれば対応可能:
- HTML/CSS直書き
- WordPress
- Wix
- その他CMS

## 技術的な設定内容

### 決済フロー

```
ユーザーがLP上の「登録」ボタンをクリック
    ↓
PayPalの決済画面にリダイレクト
    ↓
2週間の無料トライアル開始
    ↓
14日後に自動的に¥9,900/月の課金開始
    ↓
毎月自動請求（未払い時も自動リトライ）
```

### PayPal側の設定まとめ

| 設定項目 | 値 |
|---------|---|
| 製品ID | PayPal自動生成 |
| プランID | P-XXXXXXXXXX（自動生成） |
| Webhook | 必要に応じて設定（解約通知等） |
| Sandbox | 本番前にSandbox環境でテスト推奨 |

### 注意事項

- PayPalは**RFC 2047エンコード非対応**の場合がある（日本語の件名等で文字化けの可能性）
- 税金の自動計算機能はなし。税込価格で設定するのが運用上シンプル
- プランの金額変更は既存の購読者には影響しない（新規購読者から適用）
