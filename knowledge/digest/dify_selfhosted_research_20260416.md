# Dify セルフホスト版 調査レポート

## 調査日
2026-04-16

## 調査目的
Dify（AIワークフロープラットフォーム）のセルフホスト版の料金・インフラ要件・制限事項を把握する。

---

## 1. セルフホスト版のライセンス費用

### Community Edition: 無料
- オープンソース（Apache 2.0ベースのカスタムライセンス）
- ライセンス費用ゼロ
- ユーザー数・アプリ数・メッセージ数の使用量制限なし
- 商用利用OK

### ライセンス上の制約（2点）
1. **マルチテナント禁止**: 複数ワークスペースのマルチテナント環境運営は書面許可が必要（自社1ワークスペースは問題なし）
2. **ブランディング保持**: フロントエンドのDifyロゴ・著作権表示の削除・変更不可

### Enterprise Edition
- 要問い合わせ（business@dify.ai）
- SSO、高度な権限管理、専用サポート等が追加
- 価格は非公開（個別見積もり）

---

## 2. 必要インフラ構成

### Docker Composeで一括デプロイされるコンポーネント
| コンポーネント | 役割 |
|---|---|
| API Server（Flask/Gunicorn） | バックエンドAPI |
| Worker（Celery） | バックグラウンドタスク処理 |
| Web（Next.js） | フロントエンドUI |
| Plugin Daemon | プラグイン実行環境 |
| PostgreSQL | メインDB |
| Redis | キャッシュ・キュー |
| Weaviate | ベクトルDB（RAG用） |
| Nginx | リバースプロキシ |

### 最小ハードウェア要件
- CPU: 2コア以上
- RAM: 4GB以上（実運用は8GB以上推奨）

### 追加で必要なもの
- サーバー（VPS/クラウドインスタンス）: Linux推奨
- ドメイン: 外部公開する場合
- SSL証明書: Let's Encryptで無料取得可能
- LLM APIキー: OpenAI、Anthropic等（Dify自体はLLMを提供しない）

---

## 3. インフラ月額コスト

### 最小構成（個人・小規模チーム）
| 項目 | 月額目安 |
|---|---|
| VPS（2コア/4GB） | 約1,800〜3,600円 |
| ドメイン | 約100〜200円 |
| SSL（Let's Encrypt） | 無料 |
| **合計** | **約2,000〜4,000円/月** |

### 推奨構成（実運用・チーム向け）
| 項目 | 月額目安 |
|---|---|
| VPS（4コア/8GB） | 約6,000〜12,000円 |
| ドメイン | 約100〜200円 |
| SSL（Let's Encrypt） | 無料 |
| バックアップストレージ | 約150〜750円 |
| **合計** | **約7,000〜13,000円/月** |

### コスト重視（Hetzner等）
- 3コア/4GB: 約1,100円/月
- 4コア/8GB: 約2,100円/月

> LLM API利用料は別途発生（使用量に応じて変動）

## 4. クラウド版との比較

### セルフホスト版が有利な点
| 項目 | クラウド版（Sandbox無料） | セルフホスト版（Community無料） |
|---|---|---|
| メッセージ数 | 200クレジット/月 | 無制限 |
| チームメンバー | 1人 | 無制限 |
| アプリ数 | 5個 | 無制限 |
| ナレッジドキュメント | 50件 | 無制限 |
| ストレージ | 50MB | サーバー容量次第 |
| ログ履歴 | 制限あり | 無制限 |
| データの所在 | Difyクラウド | 自社サーバー内 |

### セルフホスト版のデメリット
- サーバー運用保守・アップデートは全て自己責任
- マルチテナント不可（ライセンス制約）
- Difyロゴ削除不可
- SOC Type IIレポートなし
- サポートはコミュニティ＆ドキュメントのみ（Enterprise除く）

### クラウド版料金参考
| プラン | 月額 | メッセージ | メンバー | アプリ |
|---|---|---|---|---|
| Sandbox | 無料 | 200 | 1 | 5 |
| Professional | $59 | 5,000 | 3 | 50 |
| Team | $159 | 10,000 | 50 | 200 |
| Enterprise | 要問合せ | カスタム | カスタム | カスタム |

---

## 所感
- セルフホスト Community Edition が最もコスパが良い
- インフラ費用（月額2,000〜13,000円）でクラウド版Professional（$59/月≒9,000円/月）相当以上の環境が制限なしで手に入る
- ただし運用保守の手間は自社で負う必要あり
