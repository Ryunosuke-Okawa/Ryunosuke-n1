# Claude Code / Codex 研修 LP — ビジュアル指示書

ChatGPT Image 2.0 で**セクションごとに画像生成**するためのプロンプト集。
共通のスタイル指示を冒頭でテンプレ化し、各セクション固有の内容を差分で指定する。

---

## 0. 全体共通スタイル（プロンプトに毎回入れる）

```
スタイル：
- モダンで洗練されたコーポレートLP向けのビジュアル
- フラットデザインベース、過度に装飾的にしない
- 配色：オレンジ（#FF6600）をプライマリ、白（#FFFFFF）をベース、
  ダークグレー（#1F1B16）でテキスト、淡いクリーム（#FAF6F0）でセクション地
- 余白を広く取り、要素同士に十分なスペース
- 写真より図解・イラスト主体
- 日本語テキストはサンセリフ系（Noto Sans JP相当）
- ノイズ・グラデーション・ドロップシャドウは控えめ
- ターゲット：法人の経営層・人事・DX推進担当
- 印象：「型にはめない／本気で組織を変える」「楽しく・実践的」「Anthropic公式ツールの正規法人提供者」

NG：
- ハイテク感を盛りすぎたサイバー風
- 海外SaaSによくある派手なグラデーション
- ストックフォト感の強い人物写真
- AIっぽい光るネオン演出
```

## 1. Hero（メインビジュアル）

### 用途
ファーストビューで「使ってる社員は何人？→全員を、使いこなす社員に」を一発で伝える。

### プロンプト
```
シーン：
- 横長16:9のヒーローイメージ
- 左：オフィスの会議スペースで複数の社員がノートPCを開いて議論している俯瞰イラスト
- 右：オレンジ（#FF6600）の大きな円形アクセントとCLI風ターミナル画面のモチーフ（黒背景に緑のプロンプト記号、テキストは抽象的）
- 中央：「全員を、『使いこなす社員』に。」という日本語キャッチを大きく配置できる空白
- ベース：白〜淡いクリーム（#FAF6F0）
- アクセント色：オレンジ（#FF6600）
- 印象：明るく、活気があり、研修後の理想像が伝わる
- スタイル：フラット・モダン・コーポレート

避ける：暗いトーン／コードを大写し／ロボット・AI擬人化キャラ
```

## 2. 課題提起セクション

### 用途
「知ってる」と「使ってる」のギャップを視覚化。

### プロンプト
```
シーン：
- 左右2分割の図解
- 左：「知ってる社員」— ノートPCを前にしているが手が止まっている人物が10人並ぶ（モノクロ／グレー基調）
- 右：「使ってる社員」— ノートPCで実際にタイピングしている人物が3人だけハイライト（オレンジ）
- 上部に「Claude Code を知っている社員」「Claude Code を使えている社員」のキャプション領域を空けておく
- 数字の対比が視覚的にわかるレイアウト
- ベース：白
- アクセント：オレンジ（#FF6600）／グレー（#A89F94）

避ける：人物の顔の詳細描写／表情の悲哀感
```

## 3. 「一般的なAI研修」との対比図

### 用途
3類型 vs Kawaru Team の対比を一覧で見せる。

### プロンプト
```
シーン：
- 横長の比較表ビジュアル
- 左カラム3行：「eラーニング型」「助成金パッケージ」「型にはめた集合研修」をそれぞれ象徴するアイコン
  - eラーニング：再生ボタン付きモニター
  - 助成金パッケージ：箱型の規格品・スタンプマーク
  - 型集合研修：椅子が整列した会議室の俯瞰
  - 全てグレースケール
- 右カラム3行：「Kawaru Teamの研修」を象徴する3アイコン
  - 設計：パズルピース／設計図
  - 実践：手を動かしているシルエット
  - 楽しい：ゲームコントローラー or 笑顔の集団
  - オレンジ（#FF6600）でハイライト
- 中央に縦区切り線
- 比較構造が一目で伝わるアイソメ風 or フラット

避ける：人物表情の感情強調／否定的すぎる左カラム表現
```

## 4. 3つの強み（POINT 01/02/03）

### 用途
「選べる × 楽しい × 圧倒的実績」をカード3枚で見せる。

### プロンプト
```
シーン：
- 3カード横並びのビジュアル
- カード①「選べる」：2つの料金プランカードが扇状に開いているアイコン。オレンジで「2 PLANS」のバッジ
- カード②「楽しい」：ゲームコントローラー＋ハイタッチする手のアイコン。オレンジのスパークル
- カード③「圧倒的実績」：3つの数値ラベル（5,000名／キリン／全社員）が積み重なったバッジ風
- 全カードのベース：白＋淡いクリーム
- 数字や見出しはオレンジ（#FF6600）
- 印象：「自信」「明るい」「実績重視」
- スタイル：フラットなアイコン＋シャドウ最小限

避ける：ストックフォト風／装飾過多
```

## 5. 2プラン（PLAN A セミ / PLAN B フル）

### 用途
セミ10万 vs フル15万を視覚的に差別化。

### プロンプト
```
シーン：
- 横並び2カードの料金プランビジュアル
- 左：PLAN A セミカスタマイズ — シンプルな箱型、軽快さを表すアイコン（クイックスタート系の矢印アイコン）、「100,000円/人」を中央大配置
- 右：PLAN B フルカスタマイズ — 重厚感のあるカード、レンチ・歯車・設計図アイコン、「150,000円/人」を中央大配置、右上に「RECOMMENDED」バッジ（オレンジ）
- 両カードに「10〜15時間／3〜5回／15〜25名」の共通仕様アイコン
- ベース：白
- アクセント：オレンジ（#FF6600）— 右カード（フル）の方をオレンジ強め、左（セミ）は淡いクリーム背景

避ける：価格感を否定的に見せる演出／古いExcel表風レイアウト
```

## 6. カリキュラム章立て（Claude Code / Codex 並列）

### 用途
「Claude Code研修8章」「Codex研修8章」が並列で選べることを示す。

### プロンプト
```
シーン：
- 横並び2カラムのカリキュラム俯瞰図
- 左：「Claude Code 研修」— Anthropic公式風のオレンジ系カラーで縦に8つのチャプターバッジを縦並び
- 右：「Codex 研修」— OpenAI風のグリーン or ダークグレー系で同じく縦に8つのチャプターバッジ
- 各バッジには番号と短い章タイトル（テキスト潰れOK、視認性確保）
- 中央に「OR」または「貴社が選択」の文言エリア
- 上部に「Claude Code か Codex、貴社が社内標準にしたい方を」キャプション領域

避ける：ロゴの直接使用／公式マーク模倣
```

## 7. 活用事例12個

### 用途
12業務の自動化Before→Afterをグリッド表示。

### プロンプト
```
シーン：
- 3列×4行＝12マスのグリッドビジュアル
- 各マス：業務アイコン（議事録／メール／提案書／表計算／検索／受信箱／請求書／コードレビュー／コード生成／比較表／フォルダ／日報）＋削減%バッジ
- アイコンはミニマルフラット、業務種別ごとに色相を統一せずカテゴリ別にニュートラル
- 削減%バッジのみオレンジ（#FF6600）で目立たせる
- ベース：白＋淡いクリーム
- 上部キャプション領域：「Claude Code／Codex で変わる、12 業務」
- 全体的に整然としたインフォグラフィック

避ける：実写スクリーンショット／ごちゃつき
```

## 8. 導入事例（キリン／ミズカラ）

### 用途
事例カード2枚で具体的成果を見せる。

### プロンプト
```
シーン：
- 横並び2カードの導入事例ビジュアル
- 左：キリングループ事例
  - 飲料メーカーを象徴する抽象アイコン（瓶／グラスのシルエット、ブランドロゴは使わない）
  - 中央に大きく「満足度 8.5点」「全員が業務でAI活用希望」のスタッツ
- 右：ミズカラ事例
  - 人材・組織コーチングを象徴するアイコン（円卓に集まる人型）
  - 中央に「月50時間削減」「月100時間削減」のスタッツ
- ベース：白
- アクセント：オレンジ（#FF6600）の数値強調

避ける：実際のロゴ模倣／類似の商標表現／写真合成
```

## 9. 進め方フロー（5ステップ）

### 用途
お問い合わせ→キックオフ→定着までの流れを5アイコンで。

### プロンプト
```
シーン：
- 横一列5ステップのフローチャート
- ①フォームアイコン ②吹き出し（ヒアリング） ③設計図 ④ロケット（キックオフ） ⑤伴走する人型
- 各ステップ間に矢印（オレンジ）
- 番号01〜05のラベル
- 「最短2週間でキックオフ」を④の下にキャッチ
- ベース：白
- アクセント：オレンジ（#FF6600）

避ける：時計・カレンダーアイコンの多用／古いウィザード風
```

## 11. 最終CTA（フッター直前）

### 用途
「全員を、使いこなす社員に。」を再喚起。

### プロンプト
```
シーン：
- 横長16:9のクロージングビジュアル
- 中央に丸く集まる複数の人型シルエット（笑顔・拍手・ハイタッチ）
- 周囲にオレンジ（#FF6600）のグラデーション光（穏やかに）
- 上部に「全員を、『使いこなす社員』に。」のキャッチ配置領域
- 下部に「無料相談はこちら」「サービス資料DL」のCTAボタン配置領域
- ベース：オレンジ（#FF6600）ベタ or 濃いオレンジから淡いオレンジへのグラデーション
- テキストは白（#FFFFFF）
- 印象：希望／達成／前向き

避ける：暗いトーン／単独人物／ゴール感の薄い構図
```

1. **Hero**（ファーストビュー）

6. **対比図**（一般的なAI研修との対比）
7. **カリキュラム並列**（Claude Code / Codex）

9. **課題提起**（知ってる vs 使ってる）
10. **最終CTA**
