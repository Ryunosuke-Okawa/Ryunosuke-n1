## 1. エグゼクティブサマリー

リードインクス株式会社様は、保険業界向けSaaSプロダクトを提供するソフトバンクグループ企業として、今後のAI活用が事業成長のカギとなる重要な局面にあります。

貴社エンジニアのAI活用は現状「個人の我流」に留まっており、組織としてのAI開発プロセスが確立されていません。また、海外ベンダーへの依存から脱却し、本番運用を自社で担う体制構築が急務となっています。

本提案では、貴社エンジニアがビジネス課題をAIで解決できる思考力と実装力を身につけるための**フルカスタマイズAI研修プログラム「変わるチーム」**をご提案いたします。0からのカスタム設計により、保険業界・SaaS開発という貴社固有のコンテキストに完全対応した研修を提供します。

---

## 2. 課題認識

### 2-1. 現状と背景

| 項目 | 現状 |
|------|------|
| AI活用状況 | 個人レベルでのChatGPT・Claude利用のみ。組織的なベストプラクティスが未確立 |
| 開発体制 | AI開発はPOC段階まで海外ベンダーに委託。本番運用体制が社内にない |
| ツール制約 | ソフトバンク方針によりChatGPT（エンタープライズ版）中心。セキュリティ要件あり |
| チーム構成 | 日本語話者・中国語話者が混在する多国籍エンジニアチーム |

### 2-2. 貴社が直面する3つの課題

**課題①：AI開発プロセスの知識不足**
従来の開発手法（要件定義→基本設計→詳細設計→実装）とAI開発手法の違いが理解されておらず、「ビジネス要件をAIタスクに分解する」思考が組織に存在しない。

**課題②：エンジニアのAIスキルの属人化**
ChatGPT等を個人で活用しているものの、プロダクト開発に生かせるレベルには至っていない。チームとしての標準的なアプローチが確立されていない。

**課題③：AI活用を国内自走できる体制がない**
金融業界の個人情報保護規制により、海外ベンダーへの本番運用委託は不可。社内エンジニアが自走できる能力を早急に構築する必要がある。

## 3. 提案概要

### 「変わるチーム」フルカスタマイズAI研修プログラム

#### コンセプト
**「保険業界×SaaS開発の現場で、AIを使って課題を解決できるエンジニアを育てる」**

既製の汎用研修ではなく、貴社の業務環境・ツール制約・実際のユースケースに基づいて研修内容を0から設計します。

#### 研修内容（案）

| モジュール | テーマ | 概要 |
|-----------|--------|------|
| Module 1 | AIエージェントの基礎理解 | エージェントの仕組み・設計思想。ChatGPTを超えた活用法 |
| Module 2 | ビジネス課題のAIタスク分解 | 保険業務の具体的な課題（書類OCR・支払い判断自動化等）をAIタスクに落とし込む実践演習 |
| Module 3 | AI開発プロセスの標準化 | 要件定義→エージェント設計→評価→改善のサイクル。AIプロジェクトの進め方 |
| Module 4 | ChatGPT活用のベストプラクティス | プロンプト設計・API連携・セキュリティ要件への対応 |
| Module 5 | ハンズオン実装 | 貴社の実際のユースケースを題材にしたエージェント作成演習 |

> ※上記は初期案です。ヒアリングを経て内容を完全にカスタマイズします。

#### 実施形式

| 項目 | 内容 |
|------|------|
| 形式 | 対面（エヌイチ講師が貴社オフィスに訪問） |
| 時間 | 合計10〜15時間（1日3〜5時間 × 3〜4日） |
| 対象 | SaaS本部・SIer本部のエンジニア（第1期: 15〜25名を推奨） |
| 言語 | 日本語（中国語話者にはAI翻訳ツールを併用） |
| 使用ツール | ChatGPT（エンタープライズ版）中心。貴社のツール環境に合わせて設計 |

## 4. 期待される成果・効果

### 研修終了後のゴール

- **即戦力化**: 研修終了後から、エンジニアが自社のビジネス課題に対してAIエージェントの設計・実装を自走できる状態
- **標準化**: チームとしての共通AI開発プロセスが確立され、属人化が解消される
- **内製化**: 海外ベンダー依存から脱却し、POC〜本番運用まで社内で完結できる体制の基盤構築

### 貴社ビジネスへの波及効果

1. **プロダクト競争力の向上**: AI機能をプロダクトに組み込むためのエンジニアケイパビリティが整い、保険会社・代理店への新サービス提供が加速
2. **POC案件の内製化**: 現在外部委託しているAI-POCプロジェクトを社内チームで担えるようになり、コスト削減と品質向上を実現
3. **次のステップへの布石**: AIプロダクト開発（親会社への投資申請案件）を推進できる人材基盤の形成
