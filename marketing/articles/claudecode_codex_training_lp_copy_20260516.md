| 旧版の誤り | 改訂方針 |

| Claude Code と Codex を「連携」で差別化していた | **並列扱い**。「Claude Code 研修」「Codex 研修」どちらでも選べる |

## 1. Hero

### キャッチコピー（フライヤー継承）

```
〔表（問い）〕
「知ってる社員」は何人？
「使ってる社員」は何人？

〔裏（答え）〕
全員を、「使いこなす社員」に。
型にはめない、Claude Code／Codex 研修。
```

### サブコピー

> Claude Code または Codex を、貴社の業務に組み込める社員を育てる。
> 業種・職種・リテラシーに合わせて設計する、本質を捉えた組織開発型 AI 研修です。

### CTA

- 第1CTA：`無料相談はこちら →`
- 第2CTA：`サービス資料をダウンロード`

### 信頼バー（確定）

> 累計 **5,000 名** の AI 人材育成実績 　|　 **キリングループ・ミズカラ** ほか大手研修実績多数 　|　 全社員で Claude Code を日常活用

※「累計育成人数」表現：AI スクール／法人研修を通じてこれまでに AI を教えてきた累計人数。

## 2. 課題提起（フライヤー表面のロジック継承）

> ヒーローの直下で再度問う。読者の社内を想起させる。

```
「Claude Code を知っている社員」は、たくさんいる。
でも、「Claude Code を使えている社員」は、何人いますか？

✗ ツール導入はしたが、現場で誰も使っていない
✗ 一部のエンジニアだけが触り、組織全体に広がらない
✗ ChatGPTやCopilotとの使い分けが整理できていない
✗ Claude Code と Codex、どちらを社内標準にすべきかわからない
```

## 3. 「世の中の一般的なAI研修」との対比

サービス資料 P33 の対比構造をそのまま転用。

| 世の中の一般的なAI研修 | Kawaru Team の Claude Code / Codex 研修 |
|---|---|
| **eラーニング型研修**<br>動画を見るだけ。そもそも見ない人も多く、現場の実務に結びつかない | **貴社の状況に合わせて設計**<br>本質から変えたい企業も、まず始めたい企業も対応 |
| **助成金パッケージ研修**<br>助成金ありきの汎用カリキュラム。自社の課題と噛み合わない | **現場の実務に直結する実践型**<br>貴社の実業務・実資料を題材にしたハンズオン |
| **型にはめた集合研修**<br>型通りの内容で、現場の実態とズレている | **楽しく学べる、エンゲージメントの高い形式**<br>興味のない人でも巻き込む設計 |
| | **助成金の活用にも対応可能** |

> Kawaru Team は、型にはめない研修で「本当に使える」AI 人材を育成します。

## 4. 3つの強み（看板）

フライヤー裏面の POINT 1/2/3 を完全継承。

### POINT 01　**選べる**
> セミカスタマイズ／フルカスタマイズの 2 プラン構成。
> 貴社の負担感・本気度に応じて選択できます。

### POINT 02　**楽しい**
> ゲーム形式・体験型ワーク中心の研修設計。
> 興味のない人も巻き込み、参加率と定着率を引き上げます。

### POINT 03　**圧倒的実績**
> **3 軸で証明できる、研修知見とノウハウ**
> - 累計 **5,000 名** の AI 人材育成実績
> - Claude Code・ChatGPT・Gemini・Copilot を**自社全社員が日常活用**
> - **キリングループ・ミズカラ** はじめ大手研修実績多数

## 5. 研修プログラム — 2 プラン構成

> Claude Code 研修・Codex 研修どちらにも対応。**貴社が社内標準にしたい方を選択**できます。

### PLAN A　セミカスタマイズ　**100,000 円 / 人**

> 手をかけずに、すぐ始められる。

- 業種・職種別テンプレートを土台に設計
- ヒアリングは 1〜2 回で開始可能
- 短期間・低工数で導入

**こんな企業に**：手をかけず、本質的な研修を受けたい

### PLAN B　フルカスタマイズ　**150,000 円 / 人**

> 業務・組織に最適化した、専用設計。

- 業務・課題・目標を徹底ヒアリング
- 貴社の実業務・実資料を題材にした実践型研修
- 研修後の定着・内製化まで伴走

**こんな企業に**：本気で組織の AI レベルを引き上げたい

### 両プラン共通仕様

| 項目 | 内容 |
|---|---|
| 研修時間 | 10〜15 時間 / 人 |
| 研修回数 | 3〜5 回 |
| 参加人数 | 15〜25 名 |
| 形式 | 集合 / オンライン / ハイブリッド |
| 対象 | エンジニア / 非エンジニア（クラス分け可） |
| 助成金 | 活用にも対応可能 |

## 8. 導入事例

> フライヤーに記載済みの2社をそのまま転載。

### CASE 01　キリングループ 様（飲料メーカー）

- **課題**：AI 未経験／活用イメージなし
- **対応**：業務実態をヒアリング → フルカスタマイズで体験型研修を設計
- **結果**：満足度 **8.5 点** ／ 受講者**全員**が「業務で AI を使いたい」と回答

### CASE 02　株式会社ミズカラ 様（人材・組織コーチング）

- **課題**：中間管理職に問い合わせ集中／ノウハウが属人化
- **対応**：各部門責任者のノウハウを学習させた専用 GPTs を構築・社内展開
- **結果**：営業統括 **月 50 時間削減** ／ マーケクリエイティブ **1 案件 2h → 1h（月 100 時間削減）**

## 10. FAQ

```
Q. Claude Code と Codex、どちらの研修を選べばよいですか？
A. ヒアリングで貴社の業務・既存スタックを踏まえてご提案します。両方を組み合わせる構成も可能です。

Q. ITリテラシーが低い社員でも大丈夫ですか？
A. 非エンジニア向けクラスを別途設計します。日本語の音声指示だけで Claude Code を動かせる状態を目指します。

Q. セキュリティ面が不安です。
A. エンタープライズ向けの設定（ZDR・SSO 連携）に対応。データを AI 学習に使わせない構成で運用します。

Q. オンラインだけでも受けられますか？
A. Zoom でのオンラインハンズオンに対応。集合・オンライン・ハイブリッドから選択できます。

Q. 助成金は使えますか？
A. **活用に対応可能**です。詳しくは個別にご相談ください。

Q. 受講人数の上限は？
A. 1 回あたり推奨 15〜25 名。大規模実施の場合は複数日に分割します。

Q. 研修後のサポートはありますか？
A. フルカスタマイズプランでは、定着・内製化まで伴走します。
```
