## 画像生成手法の現状整理

| 試行 | 結果 |
|---|---|
| ChatGPT Image 2.0（白ベース＋オレンジ） | 抽象的すぎてLPに合わなかった |
| HTML/CSS 実装 | デザインが微妙との判断 |
| ChatGPT Image 2.0（縦長9:16・日本BtoB LP風） | 試行中。意図と異なる結果が出ている |

### 暫定方針
3LPとも **ChatGPT Image 2.0 で縦長セクション別画像** を作る方向で、大川さん自身が ChatGPT で進める。
Claude 側はヒアリング回答テキストの提供（Step 1 / Step 3）に徹する。

## 確定事項（覚えておくこと）

1. **3LPはサブドメインで分離**して運用する
2. **顔写真は3LP全てで使わない**
3. **ロゴ画像は使わない**（社名・事例詳細はテキスト表記でOK）
4. **AI社員構築研修の自社¥6,710,000の数字はLPに出さない**
5. **画像生成は大川さんが ChatGPT Image 2.0 で実施**、Claude はテキスト指示提供のみ
6. **Claude Code と Codex は並列訴求**（どちらかを連携で立てない）
7. **LPに費用・料金は載せない**（3LP全て無料相談で個別案内）
8. 「今だけ特典」セクションは設けない（3LP共通）
