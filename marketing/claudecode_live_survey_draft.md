# Claude Code / AI活用 ライブアンケート

## 目的
- 参加者のAI活用成熟度を把握
- 課題と関心の可視化
- 営業リード化・次回コンテンツ企画のインプット

## 質問（5問構成）

### Q1. 現在のAI活用状況を教えてください。（単一選択）
- AIツールはほとんど使っていない
- ChatGPTなど対話型AIを個人でたまに使う
- 対話型AIを業務で日常的に使っている
- Cursor・Claude Codeなど開発系AIも使っている
- チーム・組織全体で導入・推進している

### Q2. Claude Codeについての現状は？（単一選択）
- 今日はじめて知った／詳しくは知らない
- 名前は聞いたことがあるが使ったことはない
- 触ってみたことはある
- 日常的に使っている
- チーム・組織で使っている

### Q3. AI活用を進める上で、今ぶつかっている課題は？（複数選択可）
- 何から始めればいいかわからない
- 学習コスト・学習時間の確保が難しい
- 社内で使える人が限られている（属人化）
- 非エンジニアへの浸透が難しい
- セキュリティ・情報管理の不安
- 業務への落とし込み方がわからない
- 導入後に定着せず使われなくなってしまう
- 費用対効果を社内に説明できない
- 経営層の理解が得られない
- 特に課題はない

### Q4. AI導入の支援を受けるとしたら、理想の形は？（複数選択可）
- 集合型の研修（対面）
- オンライン研修
- 伴走型のコンサルティング／顧問
- 個別コーチング
- ドキュメント・テンプレートの提供のみ
- 他社事例の共有
- まずは情報収集だけしたい

### Q5. 自由記述
今日の話で一番気になった点、または「もっとこの話が聞きたかった」という内容があれば、ぜひ教えてください。

## 設計意図

| 質問 | 分かること | 営業／企画への使い方 |
|------|-----------|-----------------|
| Q1 | AI活用の成熟度 | リード温度感の初期分類 |
| Q2 | Claude Code認知・利用段階 | 研修レベルの仕分け（入門 vs 応用） |
| Q3 | 組織の本音の課題 | 提案フック（どの悩みに効くか） |
| Q4 | 提供形態の好み | 研修／コンサル／セットの推し分け |
| Q5 | 未言語化のニーズ | 次回LP・コンテンツ改善 |

## 実装候補
- Google Forms + QRコード（最推奨）
- Mentimeter（リアルタイム表示で盛り上がる）
- Slido（Q&Aも同時取得）
