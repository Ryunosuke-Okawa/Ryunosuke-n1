# Claude Code / AI活用 ライブアンケート

## 目的
- 参加者のAI活用成熟度を把握
- 課題と関心の可視化
- 営業リード化・次回コンテンツ企画のインプット

## 設計方針
- 一目で読める短い選択肢
- 各質問の選択肢は3〜5個
- 心理的ハードルを下げて回答率を上げる

## 質問（5問）

### Q1. AIツールの利用状況は？（単一選択）
- 使ったことがない
- 個人で時々使う
- 仕事で日常的に使う
- チーム・組織で活用している

### Q2. Claude Codeについては？（単一選択）
- 今日はじめて知った
- 名前は知っている
- 触ったことがある
- 使っている

### Q3. AI活用で困っていることは？（複数選択可）
- 何から始めればいいかわからない
- 使える人が限られる
- セキュリティが不安
- 導入しても定着しない
- 費用対効果を説明できない

### Q4. 支援を受けるなら？（単一選択）
- 研修
- コンサル・伴走支援
- 両方セット
- まだ決めていない

### Q5. 自由記述
一番聞きたいこと・気になることは？

## 設計意図

| 質問 | 分かること | 営業／企画への使い方 |
|------|-----------|-----------------|
| Q1 | AI活用の成熟度 | リード温度感の初期分類 |
| Q2 | Claude Code認知・利用段階 | 研修レベルの仕分け |
| Q3 | 組織の本音の課題 | 提案フック |
| Q4 | 提供形態の好み | 研修／コンサル／セットの推し分け |
| Q5 | 未言語化のニーズ | LP・コンテンツ改善のヒント |

## 実装候補
- Google Forms + QRコード（最推奨）
- Mentimeter（リアルタイム表示）
- Slido（Q&Aも同時取得）
