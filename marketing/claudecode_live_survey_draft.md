# Claude Code ライブアンケート

## 目的
- 参加者の立場とClaude Code経験値を把握
- Claude Codeに対する関心・懸念を可視化
- セミナー／営業／次回LPの改善インプット

## 設計方針
- 質問はClaude Codeに特化
- 選択肢は一目で読める短さ
- 5問で完結（回答時間：約1分）

## 質問

### Q1. あなたのお立場は？（単一選択）
- エンジニア・開発者
- 非エンジニア（企画・PM・営業など）
- マネージャー・経営層
- 情シス・DX担当
- その他

### Q2. Claude Codeの利用状況は？（単一選択）
- 今日はじめて知った
- 名前は知っている
- 触ったことがある
- 日常的に使っている

### Q3. Claude Codeで気になることは？（複数選択可）
- 何ができるのかを知りたい
- 学習コスト・使いこなせるか
- セキュリティ・情報管理
- 非エンジニアでも使えるか
- チームで使う仕組みづくり
- 費用感・導入効果

### Q4. どう使ってみたいですか？（複数選択可）
- コーディング効率化
- コードレビュー・品質向上
- テスト・ドキュメント自動化
- 非エンジニアの業務自動化
- 社内ツール・プロトタイプ開発
- まだイメージがわかない

### Q5. 自由記述
Claude Codeについて、一番聞きたいこと・知りたいことは？

## 設計意図

| 質問 | 分かること | 活用方法 |
|------|-----------|---------|
| Q1 | 参加者層（エンジニア／非エンジニア比率） | 研修カリキュラムの設計 |
| Q2 | Claude Code認知・利用ステージ | セミナーの深度調整 |
| Q3 | Claude Code導入の障壁 | 営業トークのフック |
| Q4 | 活用イメージ | 個別提案の切り口 |
| Q5 | 未言語化のニーズ | 次回LP・コンテンツ改善 |

## 実装候補
- Google Forms + QRコード（最推奨）
- Mentimeter（リアルタイム表示）
- Slido（Q&Aも同時取得）
