よくあるご質問

導入前によくいただくご質問にお答えします。

QClaude Code と Codex、どちらの研修を選べばよいですか？▾
Aヒアリングで貴社の業務・既存スタックを踏まえてご提案します。両方を組み合わせる構成も可能です。
QIT リテラシーが低い社員でも大丈夫ですか？▾
A非エンジニア向けクラスを別途設計します。日本語の音声指示だけで Claude Code を動かせる状態を目指します。
Qセキュリティ面が不安です。▾
Aエンタープライズ向けの設定（ZDR・SSO 連携）に対応。データを AI 学習に使わせない構成で運用します。
Qオンラインだけでも受けられますか？▾
AZoom でのオンラインハンズオンに対応。集合・オンライン・ハイブリッドから選択できます。
Q助成金は使えますか？▾
A活用に対応可能です。詳しくは個別にご相談ください。
Q受講人数の上限は？▾
A1 回あたり推奨 15〜25 名。大規模実施の場合は複数日に分割します。
Q研修後のサポートはありますか？▾
Aフルカスタマイズプランでは、定着・内製化まで伴走します。
Qカスタマイズはどこまで可能ですか？▾
A業務題材・時間配分・対象者層まで、貴社の課題に合わせて完全カスタマイズ可能です。詳細は無料相談でお聞かせください。
