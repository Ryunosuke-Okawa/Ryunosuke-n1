# company-research — 企業リサーチ

法人営業向けに企業情報を収集・整理し、Notionへ保存・Slack用投稿文を作成するスキル。

## トリガー

以下のいずれかで起動する：
- 企業名のみを単独で送信（例：「株式会社才流」「ファインディ株式会社」）
- 「企業リサーチして」「〇〇を調べて」「〇〇を営業前にリサーチ」「〇〇の会社情報まとめて」

## ステップ1：企業名の確定

曖昧な場合（同名企業が複数存在しそう）は、国・地域・業種・代表者名などを質問して特定する。明確な場合は即座にリサーチ開始。

---

## ステップ2：Webリサーチ

`WebSearch` で以下を収集。出典URLを必ず記録する。

1. 会社概要：正式社名・設立年・所在地・資本金・従業員数・上場区分・公式HP
2. 事業モデル：主要事業・サービス・収益構造・主なブランド
3. 直近5年間ニュース：資金調達・M&A・提携など
4. 代表者経歴：氏名・経歴・方針
5. ディープリサーチ：市場ポジション・競合・財務・強み弱み・リスク・展望
6. 営業向け示唆：アプローチポイント・課題仮説・提案フック

確認できない項目は「公開情報では確認不可」と記載。

## ステップ5：Notionへの保存（OKと言われたら実行）

### 5-1. 既存ページ検索

`notion-search` で以下のバリエーションを検索：
- 株式会社〇〇 / 〇〇株式会社 / （株）〇〇 / 〇〇（株） / 社名のみ

### 5-2. 既存ページがある場合

> 「〇〇というページが見つかりました。このページに追記しますか？」

承認されたら、`### 企業情報 {toggle}` の中にレポートを追記する。

### 5-3. 既存ページがない場合

確認不要で即座に新規ページを作成。

ページ構成（デフォルトフォーマット）：

```
# 事前調査 {color="red_bg"}
### 企業情報 {toggle="true"}
    [会社概要・事業モデル・ニュース・ディープリサーチ・営業向け示唆を出典URLとセットで記載]
### 代表取締役 [氏名] {toggle="true"}
    [代表者経歴を出典URLとセットで記載]
# メモ {color="red_bg"}
### メモ１ {toggle="true"}
### メモ２ {toggle="true"}
# 議事録 {color="red_bg"}
<details>
<summary>◯月◯日　</summary>
    <details><summary>議事録</summary></details>
    <details><summary>髙橋メモ</summary></details>
    <details><summary>大川</summary></details>
</details>
```

プロパティ：MTG=正式社名、日付=今日、ステータス=商談予定

### 5-4. 保存完了 → Slack投稿文作成の確認

Notionページ保存後、ページIDを記録する。その後次を聞く：

> 「Slack用の投稿文を作成しますか？（作成のみ。送信はしません）」

## ステップ6：Slack投稿文の準備（作成する場合）

### 6-1. Googleカレンダーから日時を自動取得

`search-events` で今日から3ヶ月内のイベントを検索する。

- 検索クエリ：株式会社などを除いた社名の主要部分（例：「ファインディ株式会社」→「ファインディ」）
- 優先：タイトルに「商談」または「ZOOM」を含むイベント
- 複数ヒット：ユーザーに選んでもらう
- 見つかった場合：「「タイトル」（日時）のイベントが見つかりました。この日時で合っていますか？」
- 見つからなかった場合：手動入力を求める

### 6-2. 不足情報の確認

日時が確定したら以下をまとめて質問する：

> 「以下を教えてください：
> ① 担当者名・役職
> ② 流入経路（例：飛び込み / 紹介 / インバウンド / 展示会 など）
> ③ 課題（複数可。不明ならスキップ可）
> ④ アポ獲得者
> ⑤ コメント（任意）」

## ステップ7：Slack投稿文の出力

**Slackには送信しない。** 以下2つのメッセージ内容を画面に出力し、ユーザーがコピー&ペーストして投稿する。

### ── 親メッセージ（最初に投稿するメッセージ）──

```
【商談前情報】

日時：[日時]
企業名：[正式社名]
```

### ── スレッド返信（上のメッセージのスレッドに投稿するメッセージ）──

```
【商談前情報】

日時：[日時]
企業名：[正式社名]
名前：[担当者名]
役職：[役職]
流入経路：[流入経路]
課題

[課題1]
[課題2]

Notion：https://www.notion.so/[PAGE_ID]?v=1f8934e4905a80e1bea4000c067e5a0e
アポ獲得者：[アポ獲得者]

[コメント（あれば）]


【企業情報】
・企業名：[正式社名]
・所在地：[所在地]
・設立：[設立年]
・資本金：[資本金]
・代表者：[代表者]
・従業員数：[従業員数]
・会社HP：[URL]
・事業概要：[概要]
・主なブランド：[あれば記載、なければ行ごと省略]
```

**ルール：**
- Notion URLは `https://www.notion.so/[PAGE_ID]?v=1f8934e4905a80e1bea4000c067e5a0e`
- 資本金・従業員数は3桁カンマ区切り
- 担当者名・課題が不明な場合は「未記入」
- コメントがない場合はコメント行と前後の空行を省略

出力後、「①を投稿してから、スレッドに②を返信してください」と一言添える。
