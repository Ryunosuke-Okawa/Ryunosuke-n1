# daily-schedule — 朝の工程表生成

毎朝の工程表を作成するスキル。以下の手順で実行する。

## 実行手順

1. **予定の取得**（MCP連携済みの場合）
   - Google Calendarから今日の予定を取得（連携済みカレンダーすべて）
   - 終日予定・ミーティング・締め切りを把握する

2. **タスクの確認**
   - `DAILY.md` を読み込み、昨日の未完了タスクを確認する
   - `03_projects/` フォルダの各プロジェクトフォルダを確認し、期限の近いタスクを把握する

3. **タスクの分類**
   タスクを以下の2軸で分類する：
   - **緊急度 × 重要度**（アイゼンハワーマトリクス）
   - **認知負荷**（高 / 中 / 低）

4. **工程表の生成**
   - 15分刻みで1日のスケジュールを組む
   - 高認知負荷タスクは午前中に配置
   - バッファ時間を必ず確保する
   - 形式:
     ```
     09:00-09:15  朝のレビュー・優先順位確認
     09:15-10:30  [タスク名] ← 高集中作業
     10:30-10:45  ブレイク
     ...
     ```

5. **DAILY.mdへの書き込み**
   - 本日の工程表を `DAILY.md` の「今日の工程表」セクションに書き込む
   - 日付を更新する

## 参照ファイル

- `/Users/kyouyuu/cloude/DAILY.md`
- `/Users/kyouyuu/cloude/03_projects/` （各プロジェクト）

## 出力形式

工程表をチャットに表示してから、DAILY.mdに書き込む。
