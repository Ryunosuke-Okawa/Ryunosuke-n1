# agent-memory — メモリ保存

会話中の重要な情報をメモリファイルに永続保存するスキル。

## トリガー

「覚えておいて」「メモしておいて」「記録して」という言葉が含まれた場合に発動。

## 実行手順

1. **保存すべき情報の抽出**
   会話から以下のカテゴリに属する情報を特定する：
   - 意思決定とその理由・背景
   - ユーザーの個人的な好み・方針・スタイル
   - 事実情報・データ・重要な数値
   - クライアント情報・案件の重要メモ
   - 将来のアクションアイテム

2. **カテゴリ分類と保存先決定**

   | カテゴリ | 保存先ファイル |
   |---------|-------------|
   | 個人の好み・方針 | `00_context/memories/preferences.md` |
   | 意思決定ログ | `00_context/memories/decisions.md` |
   | クライアント情報 | `00_context/memories/clients.md` |
   | 事実・データ | `00_context/memories/facts.md` |

3. **ファイルへの追記**
   - ファイルが存在しない場合は新規作成する
   - 以下の形式で追記する：
     ```markdown
     ## YYYY-MM-DD
     - [保存内容]
       - 背景: （あれば）
       - 関連: （関連プロジェクト・クライアントがあれば）
     ```

4. **確認レポート**
   保存した内容と保存先ファイルをユーザーに報告する。
