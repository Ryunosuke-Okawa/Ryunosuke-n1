# TOMOSHIKA API 出力結果の見方

## はじめに

学生が適性検査（51問）を回答すると、システムが判定結果を返します。
本資料では、返却されるデータの各項目（フィールド）が何を意味するのかを、実際のテスト結果をもとに説明します。

---

## 各フィールドの意味と、この結果から読み取れること

### 1. result_type（判定結果の種別）

**何を表すか**: この学生がどのカテゴリに分類されたかを示します。

| 値 | 意味 |
|---|---|
| **nakanishi** | 中西製作所に適性ありと判定。中西製作所を筆頭に同系統の企業を推薦する |
| type_match | 中西製作所には該当せず、学生のタイプに合った別の企業を推薦する |
| tomocari | まだ志向が定まっていないと判定。トモキャリ（キャリア相談）へ誘導する |

**今回の結果**: `nakanishi` → **この学生は中西製作所に適性あり**と判定されました。適性検査の回答から「行動特性（主体性・変革意欲）」が高いと判断されたためです。

---

### 2. primary_type（第1位の志向性タイプ）

**何を表すか**: 6つの志向性タイプ（RIASEC）のうち、この学生が最も強く持っているタイプです。

| タイプ | 名称 | ひとことで言うと |
|---|---|---|
| R | 現実型 | ものづくり・技術が好き |
| I | 研究型 | 分析・調査が好き |
| A | 芸術型 | 創造・表現が好き |
| **S** | **社会型** | **人の支援・教育が好き** |
| E | 企業型 | リーダーシップ・挑戦が好き |
| C | 慣習型 | 正確さ・管理が好き |

**今回の結果**: `S`（社会型） → この学生は**対人支援・教育志向**が最も強い。チームの調整役やサポート業務に向いている傾向があります。

---

### 3. secondary_type（第2位の志向性タイプ）

**何を表すか**: 2番目に強い志向性タイプです。「あなたには〇〇の側面もあります」という補足情報として、結果画面に表示します。

**今回の結果**: `R`（現実型） → 社会型が主軸でありながら、**ものづくり・技術への関心**も持ち合わせている学生です。

---

### 4. primary_companies（メイン推薦企業）

**何を表すか**: この学生に最も合った企業の一覧です。中西該当の場合は、中西製作所を筆頭とした同系統企業が表示されます。

**今回の結果**:

| 企業名 | 業種 |
|---|---|
| **中西製作所** | 厨房機器 |
| キーエンス | 自動制御 |
| 平田機工 | 生産システム |
| マクニカ | 半導体商社 |
| ダイフク | 物流システム |

→ すべて**BtoB企業**が表示されています。これは、この学生のBtoB志向スコア（Q50）が高かったことを意味します。仮にBtoC志向が高ければ、ソニー・リクルート・メルカリ等のBtoC企業リストに切り替わります。

---

### 5. primary_reasons（メイン推薦理由）

**何を表すか**: 推薦企業それぞれに対して、「なぜあなたにこの企業が合っているか」を説明するテキストです。結果画面で各企業名の下に表示します。

**今回の結果**:

| 企業 | 推薦理由 |
|---|---|
| 中西製作所 | 業界シェア1位の盤石な基盤。安定した環境で「攻め」のビジネスを展開したいあなたに最適。 |
| キーエンス | 圧倒的な付加価値を生むロジック。若いうちから「稼ぐ本質」を学びたい野心家へ。 |
| 平田機工 | 世界の製造現場を支える技術力。オーダーメイドの課題解決で「一品モノ」を創る醍醐味。 |
| マクニカ | 最先端技術の目利き。商社でありながら技術力で世界を動かすスピード感を求める方へ。 |
| ダイフク | Eコマースを支える物流自動化の旗手。巨大なインフラを変革する達成感がある。 |

---

### 6. secondary_companies / secondary_reasons（補足推薦企業・理由）

**何を表すか**: 2位タイプに対応する企業グループです。「こんな企業にも適性があります」という補足として表示します。

**今回の結果**: どちらも `該当なし`
→ 中西該当（`result_type = nakanishi`）の場合、**補足企業は表示しない設計**です。中西製作所グループへの誘導を最優先するためです。

---

### 8. message（学生へのメッセージ）

**何を表すか**: 判定結果に応じた、学生への締めのメッセージです。結果画面の冒頭または末尾に表示します。

**今回の結果**: 「あなたの主体性と変革意欲は、安定基盤×ベンチャー志向の企業で大きく花開きます。」
→ 行動特性（主体性・変革意欲）が高いと判定された学生向けのメッセージです。

---

## この結果から言えること（まとめ）

1. **中西製作所への適性あり**: この学生は適性検査の行動特性スコアが高く、中西製作所に合う人材と判定されました
2. **S型（社会型）× R型（現実型）**: 人を支えることが好きで、かつものづくりにも関心がある。中西製作所の「チームで厨房機器を作り上げる」仕事と親和性が高い組み合わせです
3. **BtoB志向**: 法人向けビジネスへの関心が高いため、BtoB企業リスト（中西製作所・キーエンス等）が推薦されています
4. **中西製作所が筆頭表示**: 結果画面では中西製作所が最も目立つ位置に表示され、その後に同系統のBtoB企業が並ぶ構成になります
