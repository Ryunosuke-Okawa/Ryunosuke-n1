# TOMOSHIKA API 出力項目の説明

## 概要

学生が適性検査（51問）を回答した後、AIが判定結果をJSON形式で返却します。
以下は、返却されるデータの各項目とその意味です。

---

## 出力項目一覧

### 1. `result_type`（判定結果の種別）

学生がどのカテゴリに該当したかを示します。3種類のいずれかが返ります。

| 値 | 意味 |
|---|---|
| `nakanishi` | **中西製作所に適性あり**と判定。中西製作所を筆頭に、同系統の企業を推薦 |
| `type_match` | 中西製作所には該当せず、学生のタイプに合った企業を推薦 |
| `tomocari` | タイプが未分化（まだ志向が定まっていない）と判定。トモキャリへの誘導 |

---

### 2. `primary_type`（第1位の志向性タイプ）

6つの志向性タイプ（RIASEC）のうち、学生のスコアが最も高かったタイプです。

| タイプ | 名称 | 特徴 |
|---|---|---|
| R | 現実型 | ものづくり・技術志向 |
| I | 研究型 | 分析・論理志向 |
| A | 芸術型 | 創造・表現志向 |
| S | 社会型 | 対人支援・教育志向 |
| E | 企業型 | リーダーシップ・挑戦志向 |
| C | 慣習型 | 正確性・管理志向 |

---

### 3. `secondary_type`（第2位の志向性タイプ）

スコアが2番目に高かったタイプです。「あなたには○○の側面もあります」という補足情報として使用します。

---

### 4. `primary_companies`（メイン推薦企業）

学生に最も合った企業グループの一覧です。

- **中西該当**（`result_type = nakanishi`）の場合：中西製作所を筆頭に、キーエンス・平田機工などの同系統企業を表示
- **通常判定**（`result_type = type_match`）の場合：第1位タイプに対応する企業グループを表示
- BtoB志向 / BtoC志向に応じて、表示する企業リストが切り替わります

---

### 5. `primary_reasons`（メイン推薦理由）

メイン推薦企業それぞれについて、「なぜこの企業が合っているのか」を示す説明文です。結果画面で各企業名の下に表示することを想定しています。

---

### 6. `secondary_companies`（補足推薦企業）

第2位タイプに対応する企業グループです。「こんな企業にも適性があります」という補足として使用します。

- **中西該当**の場合は `該当なし`（中西グループのみを強調するため）
- **通常判定**の場合は、第2位タイプに対応する企業が入ります

---

### 7. `secondary_reasons`（補足推薦理由）

補足推薦企業に対する説明文です。`secondary_companies` が「該当なし」の場合、こちらも「該当なし」になります。

---

### 8. `message`（学生へのメッセージ）

判定結果に応じた、学生へのまとめメッセージです。結果画面の冒頭や締めに表示することを想定しています。

| 判定結果 | メッセージ例 |
|---|---|
| 中西該当（行動特性） | 「あなたの主体性と変革意欲は、安定基盤×ベンチャー志向の企業で大きく花開きます。」 |
| 中西該当（タイプ×志向） | 「あなたの志向性とBtoB・インフラへの適性は、社会基盤を支える企業で活かされます。」 |
| 通常判定 | 「あなたは○型の特性が最も強く、△型の側面も備わっています。」 |
| 未分化 | 「今はまだ、特定のタイプに収まる時期ではありません。トモキャリで自分の武器を見つけましょう。」 |

---

## 出力例

### 例：中西製作所に該当した場合

```
result_type       : nakanishi
primary_type      : A（芸術型）
secondary_type    : R（現実型）
primary_companies : ソニー / リクルート / メルカリ / カヤック / マネーフォワード
primary_reasons   : 各社ごとの推薦理由（5社分）
secondary_companies : 該当なし
secondary_reasons   : 該当なし
message           : あなたの主体性と変革意欲は、安定基盤×ベンチャー志向の企業で大きく花開きます。
```

> この例では、学生のBtoB/BtoC志向が「BtoC寄り」だったため、中西グループのBtoC企業リスト（ソニー、リクルート等）が表示されています。BtoB寄りの場合は、中西製作所・キーエンス・平田機工等が表示されます。
